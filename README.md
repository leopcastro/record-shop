# Record Shop

Requirements: 
 - GNU make
 - Docker 19.03.0+ 
 - Docker Compose, 
 - Port 8080 free

Separate Docker containers for Nginx and PHP-FPM

Application running on http://localhost:8080/

GNU `make` used for shortcuts of Docker and Docker Compose commands:

 - `up-build-dev` to build and start the docker containers
